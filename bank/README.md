# Bank
Simple program to demonstrate the basic operations of a bank

## Key funtionalities
- Create an account
- Close an existing account
- Modify an existing account
- Enquire balance
- Deposite money
- Withdraw money

## How to install
```
sudo chmod +x install.sh
./install.sh
```

## How to uninstall
```
sudo chmod +x uninstall.sh
./uninstall.sh
```
